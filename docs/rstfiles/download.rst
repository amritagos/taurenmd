Download
========

Download the latest version of Tauren-MD (:taurenzip:`ZIP file <>`) from the official :taurengit:`GitHub repository <>`. You can download any of the previous version `here <https://github.com/joaomcteixeira/Tauren-MD/releases>`_.

Unpack the downloaded compressed file and follow the :ref:`installation instructions <inst>` - *it's a one click install process*.
