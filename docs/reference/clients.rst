Command-line interfaces
=======================

This page documents all client interfaces available in ``taurenmd``, referred also as *taurenmd subroutines*. You may wish to read before the :ref:`Usage` page for general examples on how to use ``taurenmd`` for different purposes.


You can access a list of all *taurenmd subroutines* by running on your terminal::

    taurenmd
