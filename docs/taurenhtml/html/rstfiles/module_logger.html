logger¶

Provides a logger for Tauren-MD. Uses Python logging system.

tauren.logger.get_log(name)[source]¶

Returns a configured logger according to tauren_log_config.

tauren.logger.tauren_log_config = {'disable_existing_loggers': False, 'formatters': {'debug_format': {'format': '%(asctime)s - %(levelname)s - %(filename)s:%(name)s:%(funcName)s:%(lineno)d - %(message)s'}, 'info_format': {}}, 'handlers': {'console': {'class': 'logging.StreamHandler', 'formatter': 'info_format', 'level': 'INFO', 'stream': 'ext://sys.stdout'}, 'debug_file_handler': {'backupCount': 20, 'class': 'logging.handlers.RotatingFileHandler', 'encoding': 'utf8', 'filename': 'tauren-md.log', 'formatter': 'debug_format', 'level': 'DEBUG', 'maxBytes': 10485760}}, 'loggers': {}, 'root': {'handlers': ['console', 'debug_file_handler'], 'level': 'DEBUG'}, 'version': 1}¶
